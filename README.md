🔐 Password Generator Web Application



📝 Project Description



Password Generator is a simple web-based application that generates strong random passwords based on user preferences. Users can select password length and choose whether to include numbers and special characters. The goal is to help users create secure passwords quickly and easily.



This project is built using Python Flask for the backend and HTML, CSS, and JavaScript for the frontend.



✨ Features



Generate secure random passwords

Select password length

Option to include numbers

Option to include special characters

Clean and user-friendly interface



🛠️ Technologies Used



Python

Flask

HTML

CSS

JavaScript



▶️ How to Run the Project



🔹 Step 1: Clone the Repository



git clone https://github.com/Chami-03/password-generator.git

cd password-generator





🔹 Step 2: Create Virtual Environment (Optional but Recommended)



python -m venv venv

venv\\Scripts\\activate





🔹 Step 3: Install Flask



pip install flask





🔹 Step 4: Run the Application



python app.py





🔹 Step 5: Open in Browser



Open the following URL in your browser:

http://127.0.0.1:5000/
